This is the scaffs library for using scaffs programmatically.

If you'd like to use scaffs from the commandline look at [scaffs-cli](https://www.npmjs.com/package/scaffs-cli)
This is the cli scaffs utility.

## Install

```
npm i --save scaffs
```
